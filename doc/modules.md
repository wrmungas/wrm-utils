# Modules

### Common

**Header**: common.h

**Features**:
- provide basic sized integer types: `u8`-`u64`, `i8`-`i64`
- provide helpful macros, including:
    - `wrm_FUNCTION()` for clearer function pointer types
    - `wrm_PAIR()` for defining pair types
    - `wrm_OPTION()` for defining option types
